# Titanic Survival Prediction using Neural Networks

A machine learning and deep learning project that predicts passenger survival on the Titanic dataset.

## Overview

This project demonstrates an end-to-end predictive modeling workflow using structured tabular data. The focus is on preprocessing mixed data types and training a neural network for binary classification.

## Dataset

Source: Titanic passenger dataset

Features include:

* Passenger class
* Age
* Sex
* Fare
* Family information
* Embarkation details

Target Variable:

```text
Survived
```

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* TensorFlow
* Keras

## Workflow

### Data Processing

A preprocessing pipeline is built using Scikit-Learn components:

* Missing value handling

* Feature transformation

### Model Development

Neural network architecture built with:

* Dense layers
* Non-linear activations
* Binary classification output

* Training/validation split
