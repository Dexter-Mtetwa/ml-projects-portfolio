# Breast Cancer Wisconsin Classification

A machine learning project that classifies breast tumors as malignant or benign using the Wisconsin Breast Cancer Dataset from Scikit-Learn.

## Overview

This project walks through a complete binary classification workflow:

* Loading the Breast Cancer Wisconsin dataset
* Data exploration and preprocessing
* Train/test splitting
* Building a Logistic Regression classifier
* Model evaluation using classification metrics

The goal is to predict whether a tumor is malignant or benign based on diagnostic measurements.

## Dataset

Source: Scikit-Learn's built-in Breast Cancer Dataset

Dataset characteristics:

* 569 samples
* 30 numerical features
* Binary target variable

  * 0 = Malignant
  * 1 = Benign

## Technologies Used

* Python
* NumPy
* Pandas

## Workflow

1. Load dataset
2. Explore features and target distribution
3. Split data into training and testing sets
4. Train a Logistic Regression model
5. Evaluate performance using:

   * Accuracy
   * Classification Report
   * Confusion Matrix
